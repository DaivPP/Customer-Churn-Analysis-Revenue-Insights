#Customer Churn Analysis & Revenue Insights

##Objective
Analyze customer churn behavior for a subscription-based business and identify key drivers impacting customer retention and revenue.

##Tools & Technologies

Python (Pandas, NumPy, Matplotlib)

MySQL

Power BI

Excel / CSV

##Dataset
Telco Customer Churn Dataset (7,000+ customer records)

##Key Analysis Steps

Cleaned and transformed raw customer data using Python

Engineered analytical features such as tenure groups and charge categories

Performed exploratory data analysis (EDA) to identify churn patterns

Conducted SQL analysis to validate insights

Built an interactive Power BI dashboard with slicers and KPIs

##Key Insights

Month-to-month contracts have the highest churn rate

Customers in their first year are most likely to churn

High monthly charges increase churn risk

Value-added services like tech support reduce churn significantly

##Business Recommendations

Encourage long-term contracts with incentives

Focus retention strategies on early-stage customers

Proactively target high-risk, high-value customers

Bundle support services to improve retention
